# 翻译质量分级功能演示

## 功能演示

### 1. 速翻模式 (Fast)

**适用场景**: 日常交流、紧急处理

**特点**:
- 优化生成速度
- 保证基本忠实度和通顺度
- 简洁的输出格式

**示例**:
```
原文: "你好，世界！"
质量: 速翻
输出: 简洁的阿拉伯语翻译
```

### 2. 标准模式 (Standard)

**适用场景**: 商务、学术、内容创作

**特点**:
- 兼顾效率与质量
- 包含基本校对和润色
- 详细的翻译分析

**示例**:
```
原文: "欢迎来到我们的公司，我们致力于为客户提供最优质的服务。"
质量: 标准
输出: 
- 翻译结果
- 文本特征分析
- 专业术语识别
- 翻译建议
```

### 3. 精翻模式 (Premium)

**适用场景**: 重要文档、正式场合

**特点**:
- 追求最高质量
- 注重文化适应性
- 高质量翻译结果

**示例**:
```
原文: "这是一份重要的法律文件，需要准确翻译。"
质量: 精翻
输出: 高质量的法律文件翻译
```

## 界面操作指南

### 步骤 1: 选择翻译方向
- 点击"中-阿"按钮进行中文到阿拉伯语翻译
- 点击"阿-中"按钮进行阿拉伯语到中文翻译

### 步骤 2: 选择质量等级
- **速翻**: 点击"速翻"按钮，适合快速翻译
- **标准**: 点击"标准"按钮，适合正式文档
- **精翻**: 点击"精翻"按钮，适合重要文档

### 步骤 3: 填写翻译要求（可选）
- **翻译意图/受众**: 如"商务伙伴"、"学术交流"
- **参考译文风格**: 粘贴参考译文
- **直接要求**: 如"保留礼貌用语"、"使用正式语体"

### 步骤 4: 开始翻译
- 点击"开始翻译"按钮
- 等待翻译完成
- 查看翻译结果和分析

## 注意事项

1. **API 密钥**: 确保 Claude API 密钥配置正确
2. **网络连接**: 保持稳定的网络连接
3. **文本长度**: 长文本建议使用标准或精翻模式
4. **特殊字符**: 支持阿拉伯语和中文的特殊字符
5. **错误处理**: 系统会自动处理常见错误并提供友好提示
